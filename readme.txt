Export WooCommerce Subscriptions in CSV (Comma Seperated Values) format, with various details on the subscription, customer, and subscribed products.

== Description ==

The Subscriptions Report for WooCommerce plugin generates a CSV formatted spreadsheet containing data on subscriptions in the WooCommerce database. It is only compatible with subscriptions created by the [WooCommerce Subscriptions](https://woocommerce.com/products/woocommerce-subscriptions/) plugin.

Features:

* Select from 16 data fields that can be included in the report.
* Filter by subscription status (Active, Expired, Cancelled, etc.)

== Installation ==

1. Click "Plugins" > "Add New" in the WordPress admin menu.
1. Search for "Subscriptions Report for WooCommerce".
1. Click "Install Now".
1. Click "Activate Plugin".

Alternatively, you can manually upload the plugin to your wp-content/plugins directory.
